# Promo background music

Drop optional tracks here (MP3), then set `musicSrc` in props or in `featurePromoScripts.ts`:

| File | Suggested use |
|------|----------------|
| `background-music.mp3` | Default (also at `public/background-music.mp3`) |
| `music-calm.mp3` | Warm / story variants |
| `music-upbeat.mp3` | Hype variant |
| `music-cinematic.mp3` | Epic variant |

`musicStyle` in props still controls fade length and ducking even if you keep one file.
