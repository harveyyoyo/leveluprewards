✨ Admin quick guide

Set up rewards.
Launch in minutes.

Configure the essentials once in Admin—point categories, prize costs, and coupon rules—then teachers award and students check in at the kiosk (tap card or type ID), not the school login page.

Point categories

Name behaviors (participation, kindness) and set default values for printed coupons.

Rewards shop

Add prizes with point cost, stock counts, and who can see each item (school-wide or by class).

Coupon printing

Print 10- or 30-code sheets from Admin → Points. Students scan codes at the kiosk to bank points.

School rules

Turn on raffle tickets, kiosk timeouts, shop categories, and notifications from Settings.

How it works

1

Categories

Open Admin → Points. Create categories and default point values teachers use daily.

2

Prizes

Open Admin → Rewards Shop. Set cost, stock, and print shelf cards for the desk.

3

Go live

Teachers award points or print coupons. Students use kiosk sign-in, then scan coupon codes and redeem prizes.
